## 1. Create a Twitter Developer Account

Go to the link below and create a developer account

[https://developer.twitter.com/en/portal/dashboard](https://developer.twitter.com)


Twitter may ask you to to describe your use-cases, state that you will be using the API to publish tweets as well as reply to other tweets.

The following is an example: 


*I will be using the Twitter API to schedule my tweets and threads. This will allow me to create a consistent posting schedule where I plan on posting multiple times a week to build a following on twitter. Finally, I want to use the Twitter API to repurpose content from my other social medias on Twitter.*


## 2. Getting the API Keys

Once you create a project, make sure it is on the free tier that allows you to publish and reply to 1500 tweets a month.
